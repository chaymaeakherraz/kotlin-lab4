# kotlin-lab4
##  Description
Projet Kotlin pour le TP/Lab 4.
Ce programme simule un système de score dans un jeu en utilisant une fonction personnalisée avec plusieurs paramètres, conditions, et extensions (bonus, compteur de coups).
